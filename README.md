# Automatist Models

Downloadable AI model assets for the [Automatist](https://play.google.com/store/apps/details?id=com.automatist.app) Android app.

## Distribution

Models are distributed as **GitHub Release assets**. The app downloads them on-demand over HTTPS — no authentication required.

## Available Models

| Model | File | Size | Release |
|-------|------|------|---------|
| Gemma 3 1B IT (int4) | `gemma3-1b-it-int4.litertlm` | ~557 MB | [offline-models-v1](https://github.com/atj393/automatist-models/releases/tag/offline-models-v1) |

## Integrity Verification

Each release includes SHA-256 checksums. The app verifies file integrity after download automatically.

## Model Licensing

The Gemma family of models is licensed under the [Gemma Terms of Use](https://ai.google.dev/gemma/terms). These model files are redistributed in accordance with those terms. Google and the Gemma Authors retain all rights to the original model weights.

## Adding Models

New models are added as assets to new or existing releases. Older releases are preserved to avoid breaking existing app versions.
