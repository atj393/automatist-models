# Gemma Prohibited Use Policy (pointer)

This file is a convenience pointer. The **authoritative** Gemma
Prohibited Use Policy is the one published by Google at:

**https://ai.google.dev/gemma/prohibited_use_policy**

In case of any divergence between any copy reproduced elsewhere and the
authoritative text at the URL above, the authoritative text controls.

---

## Why this file exists

Anyone who downloads a Gemma-derivative weight file from this
repository is bound by the Gemma Prohibited Use Policy in addition to
the Gemma Terms of Use. This file exists so that the policy is
explicitly surfaced in the repository alongside the LICENSE and the
model catalogue.

> **Action for the repository owner:** optionally replace this pointer
> with a dated snapshot of the current prohibited-use policy copied
> from the authoritative URL above. Either approach is acceptable as
> long as the authoritative URL is clearly linked alongside every
> Gemma asset.
