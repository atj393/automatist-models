# Gemma Terms of Use (pointer)

This file is a convenience pointer. The **authoritative** Gemma Terms
of Use are the ones published by Google at:

**https://ai.google.dev/gemma/terms**

In case of any divergence between any copy reproduced elsewhere and the
authoritative text at the URL above, the authoritative text controls.

---

## Why this file exists

The Gemma Terms of Use require that redistribution of Gemma and
Gemma-derivative weights be accompanied by the terms. The canonical
way to satisfy that requirement is to:

1. link to the authoritative text (done above), and
2. ensure every Gemma weight file in this repository is accompanied by
   a clear notice that the Gemma Terms apply to that file
   (see [LICENSE.md](LICENSE.md) and [THIRD_PARTY_MODELS.md](THIRD_PARTY_MODELS.md)).

If Google later changes the Gemma terms, the authoritative URL above
will reflect the new terms; this repository does not freeze a specific
historical version.

## Full text

To include the full text inline, copy the complete current Gemma Terms
of Use from the authoritative URL above into this file under a clearly
marked "Full text (snapshot on YYYY-MM-DD)" heading. Keep the link to
the authoritative URL at the top of the file regardless.
